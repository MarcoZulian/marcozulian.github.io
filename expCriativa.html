Descrição

Na matéria Experiência Criativa: Navegando na Computação, pudemos desenvolver quatro projetos de programação diferentes, com as ferramentas Construct, Processing e Visual Studio.

Esses projetos permitiram um primeiro contato com lógica de programação e com ferramentas que podem ser utilizadas para concluir projetos maiores, sendo uma aula bastante prática.

Projeto 1 - Jogo

Neste projeto, utilizamos o programa Construct, para construir um jogo. Essa ferramenta funciona com drag and drop, em que a implementação das funcionalidades já estava feita, sendo apenas necessário determinar a lógica de funcionamento.

O fato de não precisarmos nos preocupar com os detalhes de implementação das funções, apenas com a lógica de funcionamento, com certeza foi um ponto muito positivo desse trabalho, já que se tratava do primeiro projeto de programação de muitos.
